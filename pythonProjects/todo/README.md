# To-Do List App

A simple command-line To-Do List app built with python
Manage tasks with due dates, mark them as completed, and save/load tasks

---

## Features

- Add new tasks with a due date
- View all tasks
- Mark tasks completed
- Delete tasks
- Save tasks to a local file
- Load tasks from a local file

---

## Running Locally

1. Clone repo:

```bash
git clone https://github.com/T-Manley/github-portfolio.git
cd python-projects/todo
```

2. Run with python todo.py

3. Follow on-screen menu to manage tasks

Task list is saved locally in tasks.dat when save option is chosen
