# Budget Tracker

A budgeting app built with Flask and a bit of SQLite, kind of like Rocket Money-ish.
You can track expenses, see totals, breakdowns by category (for now only 3, I can implement more), and filter by month.
Includes a dynamic donut chart to visualize spending by category.

[screenshots]

## Features

- Add, delete, and view expenses
- Filter expenses by month
- See total spent and spending breakdown by category
- Animated donut chart for category visualization
- Responsive and alright front-end design

---

## How to use it locally

> Uses python and flask, and requires local setup to run

1. Clone repo

```bash
git clone https://github.com/T-manley/github-portfolio.git
cd python-projects/budgeting-app
```
2. Run with python app.py

3. Use browser to look at the project --> http://127.0.0.1:5000
