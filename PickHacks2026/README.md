## PickHacksSkyWalks
SkyWalks - Automated Canopy Covered Skybridge System

## Description
Open air skybridges with reactive sun/weather coverage and illumination. Includes a formal and bulleted write up of the full idea as well as a sample code for some of the inputs affecting the automated canopy.
